Character Tags CSS Styling Guide

When this option is active, all character tags you create will be added as data HTML attributes to the .mes element. As an example, this character has two tags: Belgium and Football. Both will be applied like so: 
<div class="mes reasoning" mesid="0" ch_name="Esther De Gieter" is_user="false" is_system="false" bookmark_link="" swipeid="0" force_avatar="true" timestamp="2 de julho de 2025 22:00" data-reasoning-state="done" data-char-tags="Football,Belgium" data-char-tag-football data-char-tag-belgium></div>
Notice the highlighted data-char-tags, and both the data-char-tag-football and data-char-tag-belgium.

All tags are added inside the data-char-tags while each tag is individually added as a separate attribute. Here are two examples on how to select them with CSS:

  • .mes[data-char-tag-belgium] { color: red; } - This will select all .mes divs with data-char-tag-belgium attribute.
  • .mes[data-char-tags*="Belgium"] { color: red; } - This will select all .mes divs where data-char-tags includes the "Belgium" text. This version is not normalized and accepts spaces. Tags are split by commas if you want to select groups of tags.
  • .mes[data-char-tag-belgium][data-char-tag-football] { background-color: green; } - This will select only the .mes that have both Belgium and Football tags. This is safer than using .mes[data-char-tags*="Football,Belgium"] selector because the tag order inside the char-tag is not guaranteed.
